# 📺 Una Alienada - Monitor No Oficial (v1.0.0)
**Desarrollado por:** KirinScript (Adrián Gustavo Salvini)

>  Una herramienta diseñada para potenciar la experiencia de la comunidad de Una Alienada.

---

## ⚡ ¿Qué es esta aplicación?
Este es un **Monitor de Transmisiones** diseñado específicamente para optimizar la experiencia de la audiencia. Permite un escaneo en tiempo real para detectar cuándo comienza un directo, integra alertas visuales/sonoras, reúne todas las redes sociales en un solo panel y optimiza el consumo de recursos de tu PC (RAM/CPU) en comparación con tener el navegador abierto.

---

## 🚀 Características Principales:
* **Monitoreo Automático:** Escaneo inteligente en segundo plano. Te avisa al instante cuando arranca el stream.
* **Auto-Like (Modo YouTube):** Capacidad de dar "Me Gusta" automáticamente al iniciar un directo para apoyar el algoritmo sin que tengas que acordarte.
* **KirinBlock Integrado:** Bloqueador de publicidad nativo a nivel de red para evitar interrupciones molestas en el visor interno.
* **Modo Bandeja (Tray):** Posibilidad de ocultar la app en el reloj de Windows para que no moleste mientras trabajás o jugás.

## 🛠️ Instrucciones de Uso
* **Inicio:** Si querés que la app arranque con Windows y empiece a monitorear sola, andá a la configuración (ícono de engranaje) y marcá **"Iniciar con Windows"** y **"Auto-Iniciar Monitoreo"**.
* **Login de Google:** Para que el Auto-Like funcione, debés iniciar sesión en el botón de perfil.
* **Importante:** Los filtros de seguridad de Google son estrictos. Si te rechaza, cerrá la ventana e insistí. Una vez logueado, la sesión se guarda localmente de forma segura.
